The running procedure must be search > train > test.
Search files are : model_search.py, search_CIFAR10.py, search_CIFAR100.py
Train files are : train_CIFAR10.py, train_CIFAR100.py
Test files are : test_CIFAR10.py, test_CIFAR100.py
Visulize folders are for visualize the graph of the architectures.

The number '0' in 'search0' means this search has ran on GPU where the index if it is 3.
This directory will be made automatically based on the GPU id.

'cifar' is the name of the dataset being used.
The number after 'cifar' which is '10-20220613-185433' is the time when training was started.
